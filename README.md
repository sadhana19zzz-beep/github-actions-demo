# GitHub Actions CI/CD Demo

## Overview
This project demonstrates the basics of Continuous Integration (CI) using GitHub Actions. The workflow automatically runs whenever code is pushed to the main branch.

## Features
- Automated workflow execution
- Trigger on every push to the main branch
- GitHub Actions integration
- YAML-based workflow configuration

## Technologies Used
- GitHub Actions
- YAML
- HTML
- GitHub

## Project Structure

```text
github-actions-demo/
│
├── index.html
│
└── .github/
    └── workflows/
        └── main.yml
```

## Workflow Process

1. Code is pushed to the main branch
2. GitHub Actions automatically starts the workflow
3. The repository is checked out
4. Workflow steps are executed
5. A successful build message is displayed

## Learning Outcomes

- Understanding CI/CD fundamentals
- Creating GitHub Actions workflows
- Working with YAML files
- Automating development processes
